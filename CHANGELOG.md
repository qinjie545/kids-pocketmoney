# 更新日志

本项目的所有显著变更都将在此文件中记录。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2025-01-14

### ✨ 新增功能

- 💰 完整的零钱收入与支出记录系统
- 📊 实时余额统计和趋势图表展示
- 🔐 用户登录注册系统（支持多用户）
- 💾 SQLite3 数据库存储
- ⏰ 定时发放功能（支持每天/每周/每月周期）
- 📄 分页查看交易记录（每页20条）
- 🐳 Docker 容器化部署支持
- 📈 零钱变化趋势图表（余额、收入、支出视图）
- 🏷️ 丰富的交易分类体系
- 🎨 儿童友好的用户界面设计

### 🐛 修复

- 修复支出记录错误增加余额的bug
- 修复交易删除后余额更新问题
- 优化数据库事务处理

### 📝 文档

- 📖 完整的项目README文档
- 🤝 贡献者指南 (CONTRIBUTING.md)
- 📏 行为准则 (CODE_OF_CONDUCT.md)
- 🔒 安全政策 (SECURITY.md)
- 🐙 GitHub Actions CI/CD 配置
- 🛠️ 代码格式化和质量检查配置

### 🔧 技术改进

- 使用Flask 3.0.0框架
- 集成APScheduler定时任务
- 添加完整的自动化测试套件
- 配置代码格式化工具 (Black, isort, flake8)
- 设置Docker容器化部署

### 📦 部署

- Docker Compose 一键部署
- 生产环境Docker配置
- 详细的部署文档

## 版本命名约定

本项目使用 [语义化版本](https://semver.org/lang/zh-CN/)：

- **MAJOR.MINOR.PATCH** (主版本.次版本.补丁版本)
- 主版本：破坏性变更
- 次版本：新增功能，向后兼容
- 补丁版本：修复bug，向后兼容
